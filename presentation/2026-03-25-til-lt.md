# 私はこういうことを書いている

技術から日常の気づきまで何でも

| カテゴリ         | 例                                                  |
| ---------------- | --------------------------------------------------- |
| Computer Science | CPUアーキテクチャ、割り込み処理、hyperthreading     |
| 低レイヤー       | NASMでHello World、ポインタ演算                     |
| クラウド         | CloudFormation cfn-signal、GitHub Actions承認フロー |
| DB / SQL         | PostgreSQLの正規表現検索                            |
| その他           | VS Code設定、統計、レイヤードアーキテクチャ         |

しょうもないことでもOK、とにかく書く
