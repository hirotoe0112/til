## AWS

SSOでログインするアカウントにローカルから接続する。`https://hogehoge.awsapp.com/start`からログインする場合はSSOログイン。

1. `.aws/config`に以下の設定を追加
   ```
   [profile profile-name]
   sso_start_url  = https://hogehoge.awsapps.com/start
   sso_region     = ap-northeast-1                       # IAM Identity Centerがあるリージョン
   sso_account_id = hogehogehoge                         # アカウントID
   sso_role_name  = role_name                            # https://hogehoge.awsapp.com/startにあるロール名
   region         = us-east-1                            # メインで使うリージョン
   ```
1. コンソールで以下を実行。Webで操作を求められるため操作する
   ```
   aws sso login --profile profile-name
   ```
1. `Successfully logged into Start URL: https://hogehoge.awsapps.com/start`と表示される
1. コマンドは`aws-vault exec [profile名] -- [実行したいコマンド]`というようにするらしい
   ```
   aws-vault exec profile-name -- aws sts get-caller-identity
   ```
