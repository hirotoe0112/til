## Django経由でDBアクセス

- EC2からRDS上のテーブルのDDLが欲しかった
- EC2ではDjangoがコンテナ上で動いている
  ```bash
  sudo docker exec コンテナID python manage.py shell -c "
  from django.db import connection
  with connection.cursor() as c:
      c.execute('SHOW TABLES')
      for t in [r[0] for r in c.fetchall()]:
          c.execute('SHOW CREATE TABLE \`%s\`' % t)
          print(c.fetchone()[1] + ';\n')
  "
  ```
- 実際はターミナルが途切れるため20個ずつに分割した
  ```bash
  sudo docker exec コンテナID python manage.py shell -c "
  from django.db import connection
  with connection.cursor() as c:
      c.execute('SHOW TABLES')
      tables = [r[0] for r in c.fetchall()]
      for t in tables[0:20]:
          c.execute('SHOW CREATE TABLE \`%s\`' % t)
          print(c.fetchone()[1] + ';\n')
  "
  ```
