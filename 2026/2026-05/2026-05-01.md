## Git

- `git pull origin master`をしたら以下のエラーになった
  ```
  $ git pull origin master
  From https://github.com/hirotoe0112/til
  * branch            master     -> FETCH_HEAD
  error: cannot lock ref 'refs/remotes/origin/master': is at ba6648dd8088014aa6a60a8a11a7fe096a659a52 but expected 2362f5f7dea47a4da5b1f3f95187ef0b24af6466
  ! 2362f5f..ba6648d  master     -> origin/master  (unable to update local ref)
  ```
- AIによると、「このエラーは、ローカルのリモートトラッキングブランチ (refs/remotes/origin/master) のロックに失敗している状態です。Gitの参照(ref)が壊れているか、競合している場合に発生します。」とのこと
- 以下のコマンドを実行して解決した

  ```
  # リモートトラッキングブランチの参照を削除
  git update-ref -d refs/remotes/origin/master

  # 再度pullを実行
  git pull origin master
  ```
