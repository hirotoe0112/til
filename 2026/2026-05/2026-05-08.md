## aws-vault

- 認証情報の一覧を見る
  ```bash
  aws-vault list
  ```
- 認証情報を見る方法

  ```bash
  aws-vault exec プロファイル名 -- powershell -Command "echo $env:AWS_ACCESS_KEY_ID"
  aws-vault exec プロファイル名 -- powershell -Command "echo $env:AWS_SECRET_ACCESS_KEY"
  ```

  ```bash
  aws-vault export プロファイル名
  ```

- `ASIA`から始まるのは一時的な認証情報（STSトークン）であり、aws-vaultはセキュリティのためこちらを返す
- 長期的なアクセスキーは`AKIA`から始まるもの
