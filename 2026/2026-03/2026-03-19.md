## GitHub Actions

- サービスコンテナというのを使ってDockerコンテナを作成できる
  - [Docker サービス コンテナーとの通信 - GitHub ドキュメント](https://docs.github.com/ja/actions/tutorials/use-containerized-services/use-docker-service-containers)
- Job内で`services`に指定して起動する
- 特定の場合のみコンテナを起動したいといった場合は別Jobにする必要がある
- しかし、別Jobにすると困る場合もある（CIのチェック条件が特定のJobのみといった場合など）
- その場合、サービスコンテナを使わず、step内で直接dockerを起動することで代替できる
