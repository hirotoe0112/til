## GitHub Actions

- `$GITHUB_STEP_SUMMARY`にマークダウンを出力するとActionsのサマリー画面にレポートとして表示される
  ```
  echo "{markdown content}" >> $GITHUB_STEP_SUMMARY
  ```
- GitHub Annotation機能：特定の形式で標準出力するとActionsのサマリー画面にそのメッセージを表示させることができる。
  - また、ActionsがPRをトリガーに実行される場合はPRのその行にコメントとして表示することができる
    ```
    # 通知の場合
    echo "::notice file=app.js,line=1,col=5,endColumn=7::Missing semicolon"
    # 警告の場合
    echo "::warning file=app.js,line=1,col=5,endColumn=7,title=YOUR-TITLE::Missing semicolon"
    # エラーの場合
    echo "::error file=app.js,line=1,col=5,endColumn=7,title=YOUR-TITLE::Missing semicolon"
    ```
