## PostgreSQL UUIDv4、v7、stringの比較

- 計測フロー

  ```
  ① DROP TABLE
  ② VACUUM
  ③ CHECKPOINT
  ④ CREATE TABLE
  ⑤ EXPLAIN ANALYZE INSERT（計測）
  ```

- 使用したSQL

  ```sql
  DROP TABLE IF EXISTS users_v4;
  DROP TABLE IF EXISTS users_v7;
  DROP TABLE IF EXISTS users_v7_text;

  vacuum;
  checkpoint;

  CREATE TABLE users_v4 (
    id         UUID PRIMARY KEY DEFAULT gen_random_uuid(),
    name       TEXT NOT NULL,
    email      TEXT NOT NULL,
    score      INT NOT NULL,
    created_at TIMESTAMPTZ NOT NULL DEFAULT NOW()
  );

  CREATE TABLE users_v7 (
    id         UUID PRIMARY KEY DEFAULT uuidv7(),
    name       TEXT NOT NULL,
    email      TEXT NOT NULL,
    score      INT NOT NULL,
    created_at TIMESTAMPTZ NOT NULL DEFAULT NOW()
  );

  CREATE TABLE users_v7_text (
    id         TEXT PRIMARY KEY DEFAULT uuidv7()::TEXT,
    name       TEXT NOT NULL,
    email      TEXT NOT NULL,
    score      INT NOT NULL,
    created_at TIMESTAMPTZ NOT NULL DEFAULT NOW()
  );

  explain ANALYZE
  INSERT INTO users_v4 (name, email, score)
  SELECT 'user_' || i,
  'user_' || i || '@example.com',
  (RANDOM() * 1000000)::INT
  FROM generate_series(1, 1000000) AS i;

  explain ANALYZE
  INSERT INTO users_v7 (name, email, score)
  SELECT 'user_' || i,
  'user_' || i || '@example.com',
  (RANDOM() * 1000000)::INT
  FROM generate_series(1, 1000000) AS i;

  explain ANALYZE
  INSERT INTO users_v7_text (name, email, score)
  SELECT 'user_' || i,
  'user_' || i || '@example.com',
  (RANDOM() * 1000000)::INT
  FROM generate_series(1, 1000000) AS i;
  ```

- 100万件入れた時点でのインデックスサイズの計測

  ```sql
  SELECT
    relname                                          AS table_name,
    pg_size_pretty(pg_total_relation_size(relid))   AS total_size,
    pg_size_pretty(pg_relation_size(relid))         AS table_size,
    pg_size_pretty(pg_indexes_size(relid))          AS index_size
  FROM
    pg_stat_user_tables
  WHERE
    relname IN ('users_v4', 'users_v7', 'users_v7_text')
  ORDER BY
    relname;
  ```

  ```
  users_v4	127 MB	89 MB	38 MB
  users_v7	119 MB	89 MB	30 MB
  users_v7_text	168 MB	112 MB	56 MB
  ```
